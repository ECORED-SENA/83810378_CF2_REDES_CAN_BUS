Introducción

En este módulo se verificarán las condiciones de equipos, comprensión de diagramas de cableado y puntos de toma de las señales de CAN BUS.

1. Características de los instrumentos de medición:

a. Multímetro

Con este instrumento se miden las resistencias del circuito y realizando la medición en cada módulo se puede identificar su ubicación.

Por seguridad siempre es aconsejable usar un multímetro con una impedancia al menos de 10MΩ.

b. Equipo FSA 740

Este equipo tiene altas prestaciones debido a la posibilidad de enlazarse con otros equipos de diagnóstico, de tal manera, que se convierte en un centro de diagnóstico para todo el vehículo.

La unidad de medición es donde se agrupan todos los cables para la toma de señales del equipo, además de transformarla para enviarla al computador y de esta manera se haga visible.

Se pueden ver los cables en la parte inferior y los símbolos de cada uno de ellos, dependiendo de la conexión que se realice.

Este es un equipo analizador de motores y es por eso tantas funciones que trae y las ventajas de saber usarlo adecuadamente para aprovechar sus características.

Las conexiones de izquierda a derecha son: sensor de temperatura, conexión de batería, primario sistema de encendido, pinzas de excitación, conexiones negativas de kV sistema de encendido, conexiones positivos kV sistema de encendido, pinzas amperimétricas de 30A y 1000ª, sondas de osciloscopio 1er y 2do canal, estroboscopio, medición de presión de aire, sonda presión de líquidos.

En la siguiente imagen, se aprecian las conexiones desde la parte inferior de la unidad de medición.

Cuando se inicia a trabajar con el equipo, como primera medida se realiza la identificación del vehículo que se va a intervenir.

Se selecciona Estándar dentro de Identificación vehículo Y se despliegan las siguientes opciones

En este punto se puede escoger el tipo de motor, número de cilindros, orden de encendido y el tipo de encendido.

El equipo da 4 opciones en el tipo de encendido:

EFS con transmisor NW: Para sistemas de encendido con bobina individual por cilindro y sensor de árbol de levas.
EFS con transmisor KW: para sistemas de encendido con bobina individual por cilindro y sensor de cigüeñal.
DFS: Para sistemas de encendido DIS.
ROV Para sistemas de encendido con distribuidor

El equipo cuenta con dos opciones para tomar las señales de CAN BUS.

La primera es usando el osciloscopio universal. En esta opción, como cualquier otro osciloscopio, se hace ajuste de las variables para tener una señal que pueda ser interpretada por el técnico.

En este caso se toma la señal en los pines 6 y 14 del conector de diagnóstico y dependiendo si la señal es de baja velocidad (CAN-LS) o alta velocidad (CAN-HS), se ajusta el tiempo.

  • La amplitud se ajusta para ver una señal que se mueve entre 0 Volts y 5,0 Volts, y
  • La otra es seleccionando Prueba de componentes y luego Cable de datos CAN.

En este espacio, los ajustes para ver la señal se ajustan automáticamente, también se pueden hacer los ajustes de tiempo y amplitud.

La señal queda muy bien definida. Se pueden hacer mediciones más precisas ajustando los cursores.

Para la figura anterior, el tiempo entre los cursores registra el valor de 1,9µs. y desde 0 Volts la amplitud de la señal es de 3,61 Volts.
Se ha tomado la señal de CAN BUS-HS de tracción.
